# Bug 管理模块任务拆解

## 1. 阶段一：用户/RBAC 基础 + Bug 基础闭环

目标：完成登录、用户、权限、Bug 创建查询编辑、多图片上传和关键历史。

### 1.1 数据库

- 新增用户/RBAC 表。
- 新增 Token 表。
- 新增 Bug 表。
- 新增 Bug 附件表。
- 新增 Bug 历史表。
- 初始化基础角色和权限。

### 1.2 后端认证和用户

- 新增登录接口。
- 新增退出接口。
- 新增当前用户接口。
- 新增可指派用户列表接口。
- 新增 Token 认证拦截器。
- 新增当前用户上下文。

### 1.3 前端登录基础

- 新增登录页。
- 请求封装自动携带 Token。
- 未登录跳转登录页。
- Header 展示当前用户。
- 支持退出登录。

### 1.4 Bug 基础能力

- 新增 Bug 实体、Mapper、DTO。
- 实现 Bug 自动编号。
- 实现 Bug 创建接口。
- 实现 Bug 列表接口。
- 实现 Bug 详情接口。
- 实现 Bug 更新接口。
- 写入关键操作历史。

### 1.5 图片能力

- 新增 `BUG_IMAGE` 文件类型。
- 新增图片预览接口。
- 实现 Bug 图片上传接口。
- 校验图片类型和大小。
- 前端上传多张图片。
- 前端展示图片缩略图。

### 1.6 前端 Bug 页面

- 新增缺陷管理菜单。
- 新增 `/bugs` 路由。
- 新增 Bug 列表页。
- 支持项目、需求、状态、严重级别、指派人筛选。
- 支持新建 Bug 弹窗。
- 支持 Bug 详情编辑弹窗。
- 展示操作历史。

## 2. 阶段二：评论流 + 站内通知

目标：补齐 Bug 协作闭环。

### 2.1 数据库

- 新增 Bug 评论表。
- 新增通知表。

### 2.2 后端评论

- 新增评论接口。
- 查询评论列表。
- 评论写入操作历史。
- 评论触发通知。

### 2.3 后端通知

- 新增通知生成服务。
- 新增通知列表接口。
- 新增未读数接口。
- 新增单条已读接口。
- 新增全部已读接口。

### 2.4 前端评论和通知

- Bug 详情展示评论流。
- Bug 详情支持新增评论。
- Header 展示未读通知数。
- 通知弹窗或下拉列表。
- 点击通知定位到 Bug。
